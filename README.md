# Getting Started
## Requirements:
```
Spring Boot: 3.2.5
Maven: 3.9+
Java: 17
```

### Clone this repository:

```bash
git clone https://github.com/deepaksorthiya/spring-boot-3-solid-design-principles.git
```
```bash
cd spring-boot-3-solid-design-principles
```

### Build Project:

```bash
./mvnw clean package
```

### Run Project:

```bash
./mvnw spring-boot:run
```
